# QR-Code-Component

A clean and responsive QR Code Component built as part of a frontend development challenge.
The purpose of this project was to recreate the provided design as accurately as possible while focusing on layout structure, spacing, typography, and responsive styling.

This challenge provided a great opportunity to practice writing semantic HTML and creating polished UI components with modern CSS techniques.

---

## Overview

The project consists of a simple card component that displays:

* A QR code image
* A clear and engaging heading
* Supporting descriptive text

Although the design is minimal, the challenge emphasizes attention to detail and visual consistency across different screen sizes.

---

## Features

* Responsive and centered layout
* Clean, modern user interface
* Mobile-first design approach
* Semantic HTML structure
* Lightweight and easy to maintain

---

## Technologies Used

* HTML5
* CSS3
* Flexbox

You can also extend the project using frameworks or tools such as React, Tailwind CSS, or SCSS if desired.
